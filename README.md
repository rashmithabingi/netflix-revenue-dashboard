# 📊 Netflix Revenue & Subscriber Analysis Dashboard
An interactive Power BI dashboard analyzing global revenue and subscriber trends, built to demonstrate data visualization, KPI tracking, and business insight generation skills.

## 🎯 Overview

This project simulates a real-world Sales & Revenue Analysis use case, using Netflix-style data to track financial performance across regions, time, and subscription tiers from 2022–2024.

## 🔑 Key Insights

- **Total Revenue:** $240B+ generated across all regions (2022–2024)
- **Revenue Growth:** ~67% increase in annual revenue from 2022 to 2024
- **Top Region:** North America consistently leads in revenue contribution
- **Subscriber Base:** 3B+ total subscribers segmented across Premium, Standard, and Basic tiers
- **Profitability:** $12B in net profit against $8B in content costs

## 📈 Dashboard Features

- **KPI Cards** — Total Revenue, Net Profit, Content Cost, Total Subscribers
- **Bar Chart** — Revenue breakdown by region (North America, Europe, Asia Pacific, Latin America)
- **Line Chart** — Revenue growth trend from 2022 to 2024
- **Pie Chart** — Subscriber distribution by subscription type (Premium/Standard/Basic)
- **Interactive Slicer** — Filter the entire dashboard by year

## 🛠️ Tools & Technologies

- **Microsoft Power BI Desktop** — Dashboard creation and data visualization
- **Microsoft Excel** — Data source preparation and structuring
